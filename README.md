# simple-example-vue2-cypress-e2e

[Same example with testCafe](https://github.com/EtiennePASSOT/simple-example-vue2-testcafe-e2e)

## Project setup
```
yarn install
```

## Run tests

First, run local server :
```
yarn serve
```

Then, run cypress interface with :
```
yarn cy:open
```

or only tests with :
```
yarn cy:run
```
